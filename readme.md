# NodeJs Battleship

## Live demo
Web: https://ad-battleship.herokuapp.com/  
App Store: https://apps.apple.com/gb/app/battleshipper/id1494367531?l=nl  
Google Play: https://play.google.com/store/apps/details?id=fr.adrienaudouard.battleshipper&hl=fr

## Run front

```
cd front
npm install
npm run start
```

## Run back

```
cd back
npm install
npm run start:dev
```
